## Setup

### Frontend

Setting up the Lynnx frontend for local development is easy, it requires Git, Node.js and Yarn.

```sh
# Clone repository
$ git clone git@github.com:lukewhrit/lynnx
$ cd lynnx/frontend

# Install packages
$ yarn

# Run development server
$ yarn dev
```
